# Linux

**To download Ritchie on Linux**

* Team version

```bash
curl -fsSL https://commons-repo.ritchiecli.io/install.sh | bash
```

* Single version

```text
curl -fsSL https://commons-repo.ritchiecli.io/install_single.sh | bash
```
