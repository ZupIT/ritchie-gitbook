# Inicialização do rit

### Inicialização

Você deve executar o comando abaixo para inicializar o Ritchie: 

```text
rit init
```

Feito isso, o Ritchie adiciona o repositório de fórmulas da comunidade e cria os arquivos de configuração necessários.
